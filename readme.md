老马正在搞的线下实习项目： [http://aicoder.com](http://aicoder.com)

* 实习模式实习地址：北京市昌平区王庄工业园

* 实习方式：正规项目开发流程，大牛项目经理带，完整企业开发流程，每 5 人一个小组，项目经理控制项目进度，每个组从 3 个难度等级的项目池中挑选项目，并从无到有完成项目，项目可以做多个。

* 实习费用：标准 3 个月模式共 12000 元，1 个月模式：4500 元
